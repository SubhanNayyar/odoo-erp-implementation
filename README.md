# ERP Digital Transformation: Find My Doctor (Odoo 14)
### Capstone Project | B.S. Artificial Intelligence

A full-scale ERP implementation for **Find My Doctor**, a leading health-tech platform in Pakistan. This project involved migrating legacy Excel-based workflows into a unified Odoo 14 infrastructure to automate doctor panels, diagnostic testing, and staff operations.

## 🚀 The Challenge
Prior to this implementation, the organization managed diagnostic data, doctor panels, and patient records across fragmented Excel sheets. This led to data silos, manual entry errors, and a lack of real-time visibility for executive leadership.

## 🛠️ Technical Implementation
* **Platform:** Odoo 14.0 (Community Edition)
* **Backend Logic:** Python (.py) - Custom module development for medical data handling.
* **Frontend:** XML - Custom views, tree-views, and kanban boards for staff.
* **Database:** PostgreSQL - Relational schema design for interconnected health modules.

## 🔧 Core Contributions
* **Legacy Data Migration:** Developed automated scripts to import and clean large-scale Excel datasets into Odoo modules.
* **Role-Based Access Control (RBAC):** Configured distinct security groups and record rules for **CEO, Manager, and Staff** levels to ensure data privacy.
* **Advanced Analytics Dashboard:** Built a centralized reporting dashboard utilizing Odoo’s graph views to provide the CEO with real-time stats and business analysis.
* **Interconnected Modules:** Architected the flow between Doctor Management, Patient Records, and Diagnostic Testing modules.

## 🔜 Impact
* Eliminated manual Excel dependency, reducing data entry time by ~50%.
* Provided the executive team with a "Single Source of Truth" for business health and diagnostic tracking.
