# Capstone Project: Enterprise ERP Implementation (Odoo)
### Developed by Subhan Nayyar | Final Year Project

A comprehensive deployment and customization of an Odoo-based Enterprise Resource Planning (ERP) system to centralize business workflows and data management.

## 🚀 Project Goal
The objective of this Capstone was to replace fragmented manual processes with a unified, automated digital infrastructure. This involved mapping real-world business requirements into a scalable software architecture.

## 🛠️ Technical Architecture
* **Platform:** Odoo 16.0 / 17.0 (Community/Enterprise)
* **Backend:** Python (Custom Module Development)
* **Database:** PostgreSQL (Relational Data Modeling)
* **Environment:** Linux / Docker (Deployment Environment)

## 🔧 Core Contributions
* **Module Development:** Architected and developed custom Python modules to handle specific business logic not available in the Odoo base layer.
* **Database Optimization:** Designed relational schemas in PostgreSQL, ensuring data integrity across Sales, Inventory, and Accounting modules.
* **UI/UX Customization:** Utilized XML and QWeb templates to modify the Odoo frontend for improved user efficiency.
* **Data Migration:** Performed automated data cleaning and migration from legacy spreadsheets into the Odoo relational database.

## 📊 Key Results
* Centralized all organizational data into a single source of truth.
* Automated 40%+ of manual data entry tasks through custom logic.
* Implemented real-time reporting dashboards for executive decision-making.
